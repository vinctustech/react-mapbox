react-mapbox

A Minimalist Mapbox GL React.js Wrapper Library

This library is intended to be a thin wrapper around the Mapbox GL JS library. The only checks that are done are the normal type safety checks provided by TypeScript. You have to be careful to use the underlying Map instance correctly.
